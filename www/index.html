Weights Help

This area is provided for guidance only - Under no circumstances should these entries be relied on.

For legally accurate details use an appropriate solution i.e. public weigh bridge.

Details are automatially saved and redisplayed on entry for this device only. If you want to use more than one device then you need to replicate the details manually. Therefore, you do not need Internet access to use applicaiton and retrieve or change details

If you want to deleter details use the button.

The following table will assist in understanding how to fill in fields. Additionally, guidance is provided under each field describing how weights are calculated.

There are a number of hints to assist in filling in information and understanding assessments.

Each field has a related label that comprises of an icon and coloured text (see table below).

Inpute field backgrounds (where useful) will change depending on whether the weights are still within assessed limits and warning messages will be displayed below the field

Output fields (readonly fields) will change colours based on details entered.

Fields with soft red backgrounds highlight areas of concern (overweight). Fields with soft green backgrounds highlight weights within limits.
